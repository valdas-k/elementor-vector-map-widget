# Elementor vector Map Widget
This is a vector map widget for Wordpress Elementor website builder. It adds a vector map of Europe, various style and text settings and repeater based country pin elements, which can be expanded.

# Installation
To use this widget in Elementor, download project folder and paste it into wp-content/plugins directory. Then activate the widget in Wordpress.
